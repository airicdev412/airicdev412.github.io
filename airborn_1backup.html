Level One Updates

I've revamped the first level and completed my second run-through developing it. I went and updated a bunch of higher level things that I didn't like. I redesigned the upper platforming sections in the second half of the level. Every UV map in the level is also complete and tidy which will make working with and editing my meshes easier. I also completely reorganized the files in the project so they are easier to work with and manipulate as the project assets are growing steadily.




Still playing with skyboxes


I've changed the way the floating platforms look by retexturing the bottom of them where the dirt is. I went for a toonish layered sediment look and it looks close to how I'm envisioning it but still needs some work and maybe light detail. I've also modelled a few custom platform shapes that were needed for certain functionality in the later half of the level. Having the same repeating circular platform in different sizes was ugly and got old quickly. These new dynamically shaped platforms are tailored for each section and allows for a more coherent level.

On the main island, I've completely redone the main forested area and introduced a few "terraces". These terraces are also custom made to fit into the terrain.

I also added a watermill with a rotating platform wheel to the end of the level. The rotating platforms will provide a moving target when platforming and add a slightly different movement mechanic. One of the waterfalls will be used as a component in a gameplay obstacle and will activate the watermill.

Trees and Windmills

I began replacing the "low poly" trees that I had before with textured ones. Basically the new trees look like actual trees now and have hand drawn 2D leaf textures permeating them as opposed to the 3D looking ones before. This is the first time I've used the sapling curve plugin in Blender so I'm still learning to tweak the values to get the type of tree I'm aiming for. I am glad to have learned how to use this plugin to create these types of trees because the old ones were a bit of an eye sore and there are going to be a lot of places where these kind of trees will make the levels look much nicer.

I have also UV mapped and textured the windmill model to change it from the plain single color it had before. It's pretty basic but finally having a texture on it is letting me achieve the look I'm going for with this first level.

Catapult and Texture

Rigged the catapult and got a basic launching animation finished. Also coded up the functionality of the catapult to accomodate any rigidbody gameobjects to be launched from it once they touch the bucket. I still have to tweak some settings but soon the player should also be able to jump into the catapult and be launched far distances.
