SCREENSHOTS



VR CHARACTER CONTROLLER



  Custom character controller built for VR. Allows for different customizable state machines depending on configuration. State machines facilitate different movement states that allows for things like teleportation, dashing, flying, floating, gliding and more.



MULTIPLAYER NETWORKING


  Experience using Photon Unity Network (PUN) to create multiplayer experiences within Unity. Useful in conjuction with VR to track hmd and both hands. Syncing experiences betweeen players, maps, and monster encounters.



PROCEDURAL GENERATION


  Procedurally generating terrains for randomized maps that allow for millions of different environments to play in. Custom PCG engine facilitates multiple biomes and terrain shapes. Layering different sets of 3D perlin noise creates unique maps based on a simple seed value that can be serialized and synced across a network quickly.



PROCEDURAL ANIMATION


  Avoids static animations by coding procedural behavior for monsters for more realistic interactions.
